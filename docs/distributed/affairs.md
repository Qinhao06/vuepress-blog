# 分布式事务

**事务**是一个程序执行单元，里面的所有操作要么全部执行成功，要么全部执行失败。而**分布式事务**是指事务的参与者、支持事务的服务器、资源服务器以及事务管理器分别位于不同的分布式系统的不同节点之上。

![img](https://s2.loli.net/2023/12/04/ILUP6ce5VFkSOQN.png)

### 针对分布式理论的 CAP 理论，

- **CP** -> 刚性事务

遵循ACID，对数据要求强一致性

- **AP+BASE** -> 柔性事务

遵循BASE，允许一定时间内不同节点的数据不一致，但要求最终一致。
